Algumas Tecnologias associadas ao Desenvolvimento Web
Firefox Mozilla Firefox é um navegador livre e multi-plataforma desenvolvido pela Mozilla Foudation com ajuda de centenas de colaboradores. A intenção da fundação é desenvolver um navegador leve, seguro, intuitivo e altamente extensível
Chrome O Google Chrome é um navegador de internet,desenvolvido pela companhia Google com visual minimalista.
Interne Explorer Internet Explrer é uma série de navegadores web gráficos desenvolvidos pela Microsoft e inclusos como parte da linha de sistemas operacionais Microsoft Windows, iniciado em 1995.
Safari O Safari é um navegador desenvolvido pela Apple Inc. E incluido como navegador padrão a partir do sistema operacional Mac OS X v10.3 (Panther). Apresenta uma interface simples, característica dos produtos da Apple.
Nestscape Nestscape Navigatoe é um navegador web proprietário que foi o principal produto da NestscapeCommunicstions Corporation até 2002.
